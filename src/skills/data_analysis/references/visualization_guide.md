# 数据可视化指南

## 图表选择决策树

```
要展示什么?
├── 趋势变化 → 折线图 (line)
├── 数据对比 → 柱状图 (bar)
├── 占比分布 → 饼图 (pie)
├── 变量关系 → 散点图 (scatter)
├── 数据分布 → 直方图 (histogram)
└── 相关性 → 热力图 (heatmap)
```

## 各图表适用场景

### 折线图 (Line Chart)
- **用途**: 展示数据随时间的变化趋势
- **适用**: 时间序列数据、趋势分析
- **示例**: 日销售额走势、月度用户增长

### 柱状图 (Bar Chart)
- **用途**: 不同类别间的数值比较
- **适用**: 分类数据、排名展示
- **示例**: 各产品销量对比、地区业绩排名

### 散点图 (Scatter Plot)
- **用途**: 展示两个变量间的关系
- **适用**: 相关性分析、分布模式
- **示例**: 价格与销量关系、广告投入与转化

### 直方图 (Histogram)
- **用途**: 展示单一变量的分布情况
- **适用**: 数据分布、频率统计
- **示例**: 用户年龄分布、订单金额分布

### 热力图 (Heatmap)
- **用途**: 展示矩阵数据或相关性
- **适用**: 多变量相关性、密度分布
- **示例**: 特征相关性矩阵、时间热力分布

### 饼图 (Pie Chart)
- **用途**: 展示整体中各部分占比
- **适用**: 构成分析、份额展示
- **示例**: 市场份额、预算分配

## 可视化最佳实践

1. **选择合适的图表类型** - 不要强行使用复杂图表
2. **保持简洁** - 一个图表传达一个核心信息
3. **正确标注** - 清晰的标题、轴标签、图例
4. **颜色使用** - 避免过多颜色，注意色盲友好
5. **数据到墨水比** - 最大化数据墨水，减少装饰
